# werewolf-bot

Discord 上で人狼（人狼ジャッジメント準拠）を遊べる Bot を TypeScript で実装するプロジェクトです。ほぼ全て AI エージェントによって作られています。

## セットアップ

### 1) 依存関係のインストール

```bash
npm install
```

### 2) 環境変数

`.env.example` を参考に `.env` を作成してください。

- `DISCORD_TOKEN`: Bot トークン
- `CLIENT_ID`: アプリケーション（Bot）の Client ID
- `GUILD_ID`（任意）: 開発中は指定推奨（ギルドコマンドは反映が速い）

### 3) コマンド登録

```bash
npm run deploy:commands
```

### 4) 起動

開発（ホットリロード）:

```bash
npm run dev
```

本番ビルド:

```bash
npm run build
npm run start
```

## コマンド

### ロビー

| コマンド | 説明 | 権限 |
|----------|------|------|
| `/werewolf create [debug]` | 現在のチャンネルに村を作成します。`debug:true` でデバッグモード（役職・行動をチャンネルに公開） | 誰でも |
| `/werewolf join` | ゲームに参加します（ロビー中のみ） | 誰でも |
| `/werewolf leave` | ゲームから退出します（ロビー中のみ） | 誰でも |
| `/werewolf start` | ゲームを開始し、役職を配布します | ホストのみ |

### ゲーム進行

| コマンド | 説明 | 権限 |
|----------|------|------|
| `/werewolf skip` | 現在のフェーズのスキップに同意します。生存者全員が同意すると即時進行します | 生存者 |
| `/werewolf time` | 現在のフェーズの残り時間を確認します（自分にだけ表示） | 誰でも |

### 管理

| コマンド | 説明 | 権限 |
|----------|------|------|
| `/werewolf status [game_id]` | ゲームの現在状況を表示します | 誰でも |
| `/werewolf end [game_id]` | ゲームを強制終了します | ホストまたは管理者 |
| `/werewolf endall` | このサーバーの全ゲームを強制終了します | 管理者のみ |
| `/werewolf ping` | Bot の生存確認を行います | 誰でも |

### ボタン・UI

| UI | 説明 |
|----|------|
| 「参加する」「退出する」ボタン | ロビーの募集メッセージから join / leave 操作ができます |
| 「スキップに同意」ボタン | 昼議論・投票フェーズのメッセージに表示。`/werewolf skip` と同じ効果です |
| 投票セレクトメニュー | 投票フェーズに表示。締切前に再選択で変更できます |

## ゲームフロー

```
ロビー → 役職配布（DM）→ 夜（DM で行動入力）→ 朝（犠牲者公開）
→ 昼議論（タイマー or スキップ）→ 投票（セレクトメニュー）→ 処刑
→ 夜 → ... → ゲーム終了（勝敗発表）
```

- **夜行動**: 人狼・占い師・狩人に DM でセレクトメニューを送信。全員確定またはタイムアウトで自動解決。
- **カウントダウン**: 昼・投票フェーズで残り 60秒・30秒・10秒のタイミングにチャンネルへ通知。
- **スキップ**: 生存者全員が「スキップに同意」ボタンまたは `/werewolf skip` を使うと即時次フェーズへ進行。
