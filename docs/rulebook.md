# 人狼ボット ルールブック

## はじめに

このボットは Discord 上で人狼ゲームをプレイするための Bot です。
参加者はそれぞれ秘密の役職を与えられ、村人陣営と人狼陣営に分かれて勝利を目指します。

- 参加人数: **3〜6人**（デフォルト設定の場合）
- ゲームは1つのテキストチャンネルを拠点に進行します
- 夜の行動はボットから届く **DM（ダイレクトメッセージ）** で行います

---

## ゲームの流れ

ゲームは以下のフェーズを繰り返しながら進み、いずれかの陣営が勝利条件を満たした時点で終了します。

```
ロビー → [ゲーム開始] → 夜 → 朝 → 昼（議論） → 投票 → 処刑 → 夜 → …
```

1. **ロビー** — ホストがゲームを作成し、参加者を募ります
2. **夜** — 各役職が DM で夜の行動を入力します
3. **朝** — 夜の結果（襲撃・護衛・占いの結果）がチャンネルに公開されます
4. **昼（議論）** — 全員で話し合い、人狼と思わしき人物を推理します
5. **投票** — 処刑する人物に投票します
6. **処刑** — 最多票のプレイヤーが処刑されます（同票の場合はランダム）

---

## 役職一覧

役職はゲーム開始時にランダムで割り当てられ、本人にのみ DM で通知されます。
各役職が属する **陣営** が勝利条件に直結します。

### 村人

| 項目 | 内容 |
|------|------|
| 陣営 | 村人陣営 |
| 夜の行動 | なし |

人狼を見つけ出し、処刑することが目標です。
特殊能力はありませんが、議論・投票を通じてゲームを動かす重要な役割を担います。

---

### 人狼

| 項目 | 内容 |
|------|------|
| 陣営 | 人狼陣営 |
| 夜の行動 | **襲撃**（毎晩1人を指定） |

毎晩、村人陣営の1人を襲撃して脱落させます。
昼の議論では正体を隠しながら村人のふりをします。

- 人狼が複数いる場合、**誰が誰を選ぶかは問わず**、1つの襲撃対象が決まります
- 初日から襲撃が発生します（初日襲撃あり）

---

### 占い師

| 項目 | 内容 |
|------|------|
| 陣営 | 村人陣営 |
| 夜の行動 | **占い**（毎晩1人を指定） |

毎晩1人を選んで占い、その人物が **「人狼」か「人狼でないか」** を知ることができます。
占い結果は「人狼」または「村人」の二択で表示されます（狂人は「村人」と判定されます）。

---

### 霊能者

| 項目 | 内容 |
|------|------|
| 陣営 | 村人陣営 |
| 夜の行動 | なし（パッシブ） |

**処刑または襲撃で死亡したプレイヤーの役職** が自動的に通知されます。
夜に何かを選択する必要はありません。

---

### 狩人

| 項目 | 内容 |
|------|------|
| 陣営 | 村人陣営 |
| 夜の行動 | **護衛**（毎晩1人を指定） |

毎晩1人を選んで護衛します。その夜に人狼が同じ人物を襲撃した場合、襲撃を無効化します。
護衛した人物の名前は公開されません。

- **連続護衛禁止**: 前の夜と同じ人物を護衛することはできません
- 自分自身を護衛することはできません

---

### 狂人

| 項目 | 内容 |
|------|------|
| 陣営 | 狂人陣営（第三陣営） |
| 夜の行動 | なし |

人狼陣営の勝利を助けることが目標ですが、人狼が誰かは**わかりません**。
占い師に占われた場合、**「村人」と判定**されます。
人狼陣営が勝利した場合に狂人も勝利します。

---

## 勝利条件

| 陣営 | 勝利条件 |
|------|----------|
| 村人陣営 | 生存している人狼が **全員処刑または排除** された時 |
| 人狼陣営 | 生存している人狼の数が **村人陣営の生存者数以上** になった時 |
| 狂人陣営 | **人狼陣営の勝利**と同時に勝利 |

勝敗判定は **プレイヤーの生死が変化するたび**（処刑後・襲撃後）に行われます。

---

## フェーズ詳細

### 夜フェーズ

- 夜の行動を持つ役職のプレイヤーに Bot から DM が送信されます
- DM のドロップダウンメニューで対象を選択して送信してください
- **制限時間は3分**です。時間内に送信しなかった場合、その夜の行動は「行動なし」として処理されます（ランダムな対象には向きません）
- すべての行動が揃い次第、自動的に夜フェーズを終了します

### 朝フェーズ

- 夜の結果がチャンネルに公開されます
- 護衛が成功した場合、「〇〇は無事だった」と表示されます
- 霊能者には、処刑された人物の役職が通知されます

### 昼（議論）フェーズ

- 自由に発言して人狼を推理します
- 議論の時間に制限はありません（ホストが任意に投票フェーズへ進めます）

### 投票フェーズ

- 全員が1票ずつ、処刑したいプレイヤーに投票します
- **記名投票（デフォルト）**: 誰が誰に投票したかが公開されます
- **匿名投票（設定変更時）**: 投票先のみ公開され、投票者は非公開です

### 処刑フェーズ

- 最多票を獲得したプレイヤーが処刑されます
- 最多票が複数いる場合、その中からランダムで1人が処刑されます
- 処刑されたプレイヤーの役職は **全員に公開** されます
- 処刑後に勝敗を判定し、決まらなければ次の夜フェーズへ進みます

---

## 特別ルール

- **初日から行動あり**: 1日目の夜から人狼の襲撃・占い師の占い・狩人の護衛がすべて発生します
- **自分自身への行動不可**: どの役職も自分自身を対象に選ぶことはできません
- **夜行動のタイムアウト**: 制限時間（3分）を過ぎた場合、未提出のアクションは「行動なし」扱いになります。ランダムな対象が自動選択されることはありません
- **連続護衛禁止**: 狩人は前の夜と同じ人物を護衛できません

---

## コマンド一覧

すべてのコマンドは `/werewolf` から始まります。

| コマンド | 説明 |
|----------|------|
| `/werewolf create` | ゲームを作成してロビーを開始します |
| `/werewolf join` | 現在のロビーに参加します |
| `/werewolf leave` | ロビーから退出します |
| `/werewolf start` | ゲームを開始します（ホストのみ） |
| `/werewolf status` | 現在のゲーム状況を表示します |
| `/werewolf end` | ゲームを強制終了します（ホストまたはサーバー管理者） |
| `/werewolf endall` | このサーバーの全ゲームを終了します（サーバー管理者のみ） |
| `/werewolf rules` | このルールブックを表示します |
